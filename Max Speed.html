Max Speed

Description

This is a game I have in development. The core mechanic of the game is that the players speed increases depending on how many coins they have collected. This mechanic was inspired by Sonic Unleashed on Wii where they player collected rings and their speed increased. I have been working on new mechanics and there is still more to be added. But I have a playable build that can be found on my itch.io page.
